# Contributing

Thanks for your interest in improving Claude Memory.

## Ways to contribute
- Bug reports
- Documentation improvements
- Setup clarification for different platforms
- Feature suggestions
- Pull requests

## Before submitting a PR
- Keep changes focused and small
- Update README/docs if behavior changes
- Describe the problem being solved and the expected result

## Good first contributions
- Improve setup docs
- Add examples or screenshots
- Improve error messages
- Add optional multi-user support notes
